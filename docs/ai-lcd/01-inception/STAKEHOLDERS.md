## Necesidades por Stakeholder

### Diego (Admin)

- Subir PDFs de noticias de forma rápida
- Crear cuentas para familiares
- Monitorear que el sistema funcione
- Poder usar OpenAI API (ya tiene API key) para respuestas de alta calidad
- Que el sistema sea mantenible y documentado (AI-DLC)

### Familia (Users)

- Entrar a una página web como cualquier otra (no instalar nada)
- Login simple con usuario/contraseña
- Escribir preguntas en lenguaje natural y obtener respuestas claras
- No poder romper nada por accidente (no pueden subir, borrar ni configurar)
